Key Features of Bootstrap


Grid System

Containers, rows, and columns together can be used to create a grid system
Columns are automatically divided equally like this.
Borders can also be added to said columns
There can be a max of 12 columns per row.

Text Display

  • the above text is a display heading
  • lead can be used to make a paragraph stand out
  • tag mark is used to highlight text
  • tag del is used for deleted text (strikethroughs)
  • class text-muted results in lighter text
  • tag small results in fine print
  • tag abbr can be used to show text when you hover over it
  • class badge can be used to add badges
